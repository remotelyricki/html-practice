Text and Miscellaneous Tags


P Tag

The p tag is used often in HTML for writing a block of regular text. This block of text is currently being written in a p tag.

<p>This is a regular p tag.</p>

The output for this code will look like:

This is a regular p tag.


Strong Tag

If you would like your words to appear bold, you need to use a strong tag.

<p><strong>This sentence will appear as bold text.</strong></p>

The output for this code will look like:

This sentence will appear as bold text.


Em Tag

If you would like your words italicized, you will need to use the em tag.

<p><em>This sentence will appear as italicized text.</em></p>

This sentence will appear as italicized text.


Mark Tag

If you would like your words highlighted like I have been using for the code on this webpage, you will need to use the mark tag.

<p><mark>This sentence will appear as highlighted text.</mark></p>

This sentence will appear as highlighted text.


Small Tag

If you would like your words to appear small, you will need to use the small tag.

<p><small>This sentence will appear as highlighted text.</small></p>

This sentence will appear as small text.


A Tag

An a tag lets us use hyperlinks to connect one page to another. You use the href attribute to assign the hyperlink to the tag. If yous use a target attribute to equal _blank, it will open the hyperlink in a new tab when clicked. If not it will redirect from your page to the new one.

<a href="https://rubberduckdebugging.com/" target="_blank">Rubber Duck Debugging</a>

Rubber Duck Debugging

Buttons

A button can be used in conjunction with JavaScript or some other language to perform an action, but for now we are just going to make a basic HTML button that does nothing except look like a button.

<button>Click Me</button>



Unordered Lists

An unordered list is a list of words that appear as a bullet point list.

People most often use unordered lists to make their navigation bars using CSS to style them.

  • Duck
  • Snake
  • Cat
  • Dog

Here is how you code it:

<ul>

 <li>Duck</li>

 <li>Snake</li>

 <li>Cat</li>

 <li>Dog</li>

</ul>



Ordered Lists

An ordered list is a list of words that appear as a numbered list.

  1. A
  2. B
  3. C
  4. D

Here is how you code it:

<ol>

 <li>A</li>

 <li>B</li>

 <li>C</li>

 <li>D</li>

</ol>

Forms

There are many different styles you can use in your forms tag. Here are some examples:

When you want text to return, put the input type set equal to text.

<form>

 <label>Name</label>

 <input type="text" name="userName" placeholder="John Doe">

</form>



When you want a valid email, set the type as email so you will return a valid email or it will error out before the user can send it.

<form>

 <label>Email</label>

 <input type="email" name="userEmail" placeholder="jdoe@something.com">

</form>



A phone number would be very similar to a name.

<form>

 <label>Phone Number</label>

 <input type="text" name="userNumber" placeholder="123-456-7890">

</form>



For longer areas of text, instead of input as your tag, you will want to put textarea.

<form>

 <label>Message</label>

 <textarea name="message" placeholder="Write your message here..."></textarea>

</form>



We can also include options to choose from within a form.

<form>

 <select name="bestContact">

  <option value="email">Email</option>

  <option value="call">Call</option>

  <option value="text">Text</option>

 </select>

</form>



If you would like a date, HTML has a calendar that you can make appear by making the input type equal to date.

<form>

 <label>Birthday</label>

 <input type="date" name="birthday"></input>

</form>



If you add the input type as submit, it will create a button that you can assign value to using backend code.

<form>

 <input type="submit" name="submit" value="Submit">

</form>



Images

You should have images saved locally so you can just call to them from a folder.

If they are not saved in a folder, you can put a hyperlink in the src attribute to link to an external image.

<img src="duck.jpg" alt="Rubber Duck" width="200">



Quotations

The first way to quote someone is using a blockquote. In a clockquote, you will just see the text on the screen. Even if you site to source in your code, it will not show up unless you pull up the code to see it.

Lorem ipsum

<blockquote cite="google.com">Lorem ipsum</blockquote>


The next way is to have an abbr tag so that you are able to hover over a text and see the citation or comment.

I am learning HTML

<p>I am learning <abbr title="HyperText Markup Language">HTML</abbr></p>


The final way that I will talk about is the cite tag.

Hello World RemotelyRicki

<p><cite>Hello World</cite> RemotelyRicki</p>



Commenting in HTML

In HTML and pretty much all other coding languages, you have the option to put comments in your code. This is text that you can see in your code, but it doesn't show up on the webpage that you are creating.

This can help a developer organize their work by titling each section of code, explaining what they are trying to attempt with the code, or even provide a placeholder with a description of what they intend to do in a particular section.

A comment in HTML appears in between these symbols <!-- -->

For Example:

<!-- This section has to do with commenting in HTML -->
